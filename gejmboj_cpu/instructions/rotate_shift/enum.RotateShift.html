Expand description

Bit rotate and shift instructions.

Some instructions operate on m which is indicated by the 8-bit operand passed to the instruction as per the following table:

OperandTarget
0000_0110(HL), the memory contents pointed to by the HL register
0000_0rrr8-bit register rrr
